5 HTML tags
<h1:>
The heading tags are used to mark headings according to their importance. This tag stands for the most important heading of the web page.
<p:>
Defines a paragraph of text. It is a block-level element and always starts on a new line. Before and after each paragraph, browsers add margin automatically.
<q:>
Used to mark up quotes embedded in other elements like a paragraph.
<strong:>
Lots of emphasis, strong importance or urgency
<i:>
Another language, technical term, title. Visual only italics.
